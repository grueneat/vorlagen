01Strategy

Read this first — the rest is detail.

Strategy summary

Gotham Narrow is fully removed from all 16 Scribus templates and replaced by Barlow Semi Condensed (committed OFL, rendered locally for the print pipeline). Vollkorn stays as the accent/emphasis face — its genuine static italics (Black + Bold) are now vendored reproducibly. The Schriftvergleiche feature is removed, and all 16 baseline PDFs were regenerated as the new comparison target after a full visual sign-off.

  • Two-font, design-system-aligned result.

    Barlow for everything that was Gotham; Vollkorn for italic accents/pull-quotes (= org DS --gat-font-headline/copy + --gat-font-emphasis). pdffonts proves ONLY Barlow + genuine Vollkorn embed — zero DejaVu/Gotham/Minion/Times.

  • Visual sign-off gated the baselines.

    All 16 templates reviewed; two objective text-overflow defects (Barlow's narrower metrics) fixed at source and re-rendered to zero dropped characters before any baseline was promoted.

  • Reproducible font provisioning.

    Barlow + genuine Vollkorn committed under fonts/ with working .gitignore exceptions; the Docker build installs them and skips any competing drop-zone Vollkorn — no substitute font can sneak in.

03Decisions

Key decisions & options

Vollkorn stays — only Gotham replaced

User correction

"Die Vollkorn Schrift muss bleiben, nur gothic muss ersetzt werden." The earlier all-fonts→Barlow pass was reverted for Vollkorn only (55 build.py refs + 3 originals + ci.yml), attribute-anchored against the pre-swap state — Gotham-origin Barlow runs untouched.

Genuine Vollkorn vendored reproducibly

User correction

"Keine andere Schrift, installiere die richtige vollkorn dazu." The real FAlthausen OFL static Vollkorn-BlackItalic.ttf + Vollkorn-BoldItalic.ttf are committed under fonts/vollkorn/; a competing variable-font substitute was caught and excluded; the drop-zone install now skips Vollkorn-*.

Baseline source = template.sla, not preview.pdf

Correctness fix

visual_diff renders template.sla; the 3 original_sla templates' preview.pdf comes from a divergent template-preview.sla. Baselines rendered from template.sla diff at 0.0000% (preview-derived drifted 6–46%).

No tolerance loosened

Empirical

Scribus rasterises deterministically (byte-identical re-renders); measured drift is 0.0000% per page, so the existing tight budgets stand — nothing was widened to pass.

Out of scope (pre-existing)

templates/_smoke/* fixtures and IDML-provenance scaffolding docs still name Gotham — they record the immutable InDesign source for the audit oracle and are untouched by this branch.
